VPS 这个东西真的是个宝藏，越用越能发现新东西，还能学习到新知识。生活质量也是提高了不少。

虽然说了无数遍了，但我还是要说一遍，购买 VPS 搭建博客真的是我这两年做的最正确的一件事

所以这篇文章就来分享我都用 VPS 做了哪些事情，如果你有 VPS ，欢迎交流分享，如果你没有，说不定这篇文章就是剁手的理由了。

博客

最开始自然是搭建博客了，关于为什么搭建，以及博客给我带来了什么，你可以看看 这篇文章

如何搭建博客，你可以看看 这篇文章

梯子

梯子是一把钥匙，能打开一扇更加无穷知识的大门。当然首先英语还是得好好学，要不然还是有点难受的

BT下载

因为国内的网大多是内网 ip ，获取到公网 ip 也是困难重重，但是 BT 下载就非常需要 公网 ip ，VPS不仅是公网 ip ，下载速度还非常高（国内 VPS 除外）。所以自从我用了 VPS 上的 BT 下载后，就真的没担心过找资源啥的了，所有片子蓝光随便看。

为了可持续发展，一定要保种哦

私人云盘

不仅能摆脱百度云的限速，还不用担心小姐姐变成八秒视频，分享也很方便。我已经分享过好几次给周围朋友了，效率比百度云不知道高到哪里去了。

除此之外，自动上传备份也多次救我于水火。

关于如何搭建我也写过 文章

Telegram RSS机器人

这个也是极大提升了我的使用体验，RSS 是有很多客户端的，但是体验都不怎么样，推送也不行。RSS 我主要也是接收一些资讯推送，不是阅读文章为主，多个客户端成本就很高。

但现在有了 Telegram bot，上面的一切问题就没有了，自建的机器人可以自己设置刷新时间，Telegram 的消息推送可以使用tele graph 直接阅读文章，真的非常牛逼。相信我，这真的就是用了就回不去的体验。

除此之外，Telegram bot非常强大，只要你想，辅以相关知识，你可以实现你想实现的任何功能。

自动化脚本

比如之前我写过的 自动下载B站收藏视频 ，还有其他的一些脚本如监控我的一些服务运行状态，有问题自动发邮件提醒。

这些都是能极大地提升生活质量的。

最后

目前我就做了这么多，感觉 VPS 就像一块画板，想要画什么，完全取决于你，拥有着无限的可能性。

之后我应该还会继续探索，发掘出更多玩法
