工作前我的一个想法就是，以后每年都换手机，体验最新科技；但到了真正工作的时候，反而舍不得了

今年的 Pixel 7 Pro 各方面相比我的 Pixel 5 都算是巨大提升了：

  • 2K 屏幕，120HZ，LTPO
  • 屏下指纹（我还从未使用过屏下指纹
  • 12G RAM
  • 更好的性能

所以在发布后虽然价格很高，我仍然打算在发了年终奖之后购买；但前天的一条新闻把我吓到了，我想了想早晚都要买，所以马上就下单了

开箱



标准 Pixel 包装，没有充电头

另外因为这个手机太大了，怕被摔又下单了一个原装手机壳


手机壳手感很好，贴合完美，按键清脆

上手体验

第一感受就是太大了

用了一天了还没适应，比我上上台 2 XL 还要大一点点

屏下指纹体验还行，也是还没适应，老是下意识摸屏幕后面；另外用的时候指纹区域亮度会拉满，黑暗环境下用会闪下眼睛

120 HZ 没啥感觉，2K 也没啥感觉；但总体上感觉还是比 Pixel 5 流畅了一些

另外 root 的时候 patch 不是 patch boot.img，而是 init_boot.img

对我来说就是一次例行升级，应该不会写详细体验文章，相机我也用的少；Twitter 上可能会分享一些使用体验，不过还不如去看评测博主的

最后再碎碎念一下，买 Pixel 其实几乎已经成为我的唯一选择了，完全没有广告，Root 没有任何限制，系统纯净，可以完全掌控，不用担心内置反诈中心什么的；

这也是我愈发反感手机评测的一个原因，一个手机评测，一半时间测性能，一半时间说相机；根本就没说真正实际的使用体验。

当然这也是我一家之言，对于普通消费者来说，Pixel 确实是极其难用的，国产手机是更好的选择；但如果你注重隐私，讨厌臃肿的系统和广告，并且愿意折腾的话（并不用一直折腾，刚用的时候设置好后续就正常用了（倒是可以水篇文章讲讲我是怎么用的，Pixel 应该是最好的选择了
