最近又开始折腾 Tasker 了，主要是之前使用总有些地方用着不顺手，于是想着用 Tasker 改善下，体验了一个多月了，感觉挺不错的，分享下我做的一些功能。

使用前准备

为了保证我介绍的功能你都能使用，需要先做些准备：

  • Root
  • 将 Tasker 改为系统应用

将 Tasker 改为系统应用可以使用 Magisk 的 App Systemizer 模块做到。

省电：点击 首选项，选择 监视器，将 使用可靠计时器 改为 从不，可以做到让 Tasker 几乎和没装之前一样。

自动开启黑夜模式并切换动态壁纸

首先自动开启黑夜模式，这很容易做到，根据时间或者外界亮度这都可以

难点就在自动切换动态壁纸，这个我也不会，好在有人在 reddit 上问了，作者做了一个，在手机上点击这个 链接 ，就能导入配置文件到 Tasker

这样就解决问题了，具体切换哪张壁纸取决于这个值，是多少就需要你自己测试了

切换就设置为当检测到 Dark mode 开启时切换

同时 Tasker 开启 Dark mode 需要额外设置，打开手机 ADB ，连接至电脑，复制以下命令到 cmd 执行

adb shell pm grant net.dinglisch.android.taskerm android.permission.WRITE_SECURE_SETTINGS

效果

微信/支付宝 支付完自动跳转到记账界面

在一年多以前我就开始了记账，但老是忘记，所以有了这个功能，实现也很简单，微信和支付宝的支付之后都有个固定的支付标题，检测这个即可。

打开记账不仅仅是打开软件，而是直达记账的那个界面

这个命令很好理解，am start -n 就是打开 Activity，即记账页面

com.mutangtech.qianji ，就是要打开的软件的包名， com.mutangtech.qianji.bill.add.AddBillActivity ，这个就是记账的页面，这里我使用的是 钱迹 这个软件。

具体的查看包名，和 Activity ，可以使用 开发者助手 这个软件查看

效果

自动记录付款数字其实我也做了，支付宝的通知是包含支付数字的，只要用正则表达式匹配就好了，微信不显示数字就很麻烦，我没想到该怎么做，钱迹开发者说新版本会支持自动记账，看看吧。

横屏禁用通知弹出

因为是原生系统，看视频时弹出的通知总是挡住内容。这个功能的作用就是手机横屏时，即视频/游戏时关闭通知弹出，但仍接收通知，这样结束视频后，能看接受的消息。

原本我想用勿扰模式来做这个功能，但不知道为什么就是不行，于是采用了另一个办法：shell命令

执行这个命令就能关闭通知弹出，后面的0改成1就是打开

settings put global heads_up_notifications_enabled 0

最后

其实还有一些我做的没讲到，但都很简单，所以挑了这几个在做的时候困扰到我的来讲，Tasker 是个非常牛逼的软件，但非常可惜的是用的人没几个，所以它一直不温不火，希望还是多点人用，这样才能有更多的创意被分享出来。
