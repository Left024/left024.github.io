接触 VPS 快一年了，也算是体验了蛮多家 VPS 了，是时候写一篇小小的使用体验了；应当是有那么一点参考价值的。

Vultr

我第一家买的就是它，有一说一 vultr 是最适合 VPS 初学者的，按小时计费，不喜欢就删，完全不需要去找客服。

可选的机房地点很多，但是网络线路非常传统，不适合爬墙，尤其是晚上高峰期几乎不可用

系统非常全面，是我目前为止见过最全面的

而且它的预装系统是我用的最舒服的，如果你是初学者，用 vultr 的机子按照网上的一些教程（博客，ss等等）一步步做基本不会出错。而其它商家我就经常性的出问题，往往要自己去找原因折腾半天

总结一下，vultr 的使用成本是很低的，系统支持非常全面，非常适合初学者

搬瓦工

搬瓦工是我现在博客在用的，搬瓦工也是很有名气，随便搜搜都是推荐它的，选择它也是因为年付比 vultr 便宜，网络线路好了一点，再加上我也算是从小白到大白了，基本知识使得我可以应对可能出现的问题。

对比 vultr 搬瓦工缺点还蛮多的：支持的系统不多，机房只有洛杉矶，换 ip 麻烦

优势就是cn2 或 cn2 gia线路，以及相对便宜的价格了

Hostsolutions

买这家主要是 搭建私人云盘 ，他家的无版权，抗投诉蛮出名的，而且价格低廉

但是缺点同样明显：垃圾配置，使用极其卡顿，安装极其缓慢，只有罗马尼亚 ip

当然这些只基于 storage plan，其它方案我没用过就不评判了

在网盘使用方面，你可以看看 这篇文章

CrownCloud

因为 hostsolutions 使用过于蛋疼，所以我买了这家，配置相对来说比 hostsolutions 好点，网络也好点

但是它并不抗投诉

所以收到投诉后果断申请退款

退款效率还是不错的

HostDare

买这家是为了改善我的爬墙体验，网站介绍是对中国有专门的优化

然并卵，刚买完 ip 是不可用的，只能开工单申请换ip

虽说是免费更换 ip ，但是你看看我等了多久

然后我配置完 ss 测试，和搬瓦工的没有啥区别，甚至还感觉有点差

于是我果断申请退款，又是漫长的等待：

好在还是退了，强烈建议不要购买这家，抛开配置网络不谈，这个工单速度真是磨死人

Akkocloud

最后选择了这家，国人开的，价格很诱人

不过考虑到之前的情况，我先买了一个月试试，速度确实很棒，线路很优秀。

在买的一个月过了之后果断年付 288（现在涨价了），算下来一个月不到30，四舍五入等于不要钱。

因为只是用来爬墙，所以配置啥的完全够用。

唯一的缺点就是因为过于便宜，经常被攻击，我买了两个月已经被打了三次了，不过老板响应蛮快的，一个小时内都能修复。
