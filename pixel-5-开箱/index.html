前言

作为一个老谷歌用户，每代的 Pixel 我都会关注，这次也不例外，而且今年我的 Pixel 2XL 烧屏也越来越严重（如图为纯白背景下）

再加上 Ram 也不够用了，开多个软件就很容易卡顿，让我萌生了换新的念头。

然而这次的 Pixel 5 怎么看都像个智商税，高通骁龙 765G ，90 Hz 1080P 的屏幕，8+128，这么个配置卖 699 刀。

有这钱买 Pixel 4 XL 它不香吗？

刚开始我也是这么想的，但当我看到 Pixel 5 的实际上手时，我被它的外观深深的吸引了。

真的太好看了，尤其是这个视觉上的四边等宽，小巧的机身

智商税又怎样？千金难买我乐意；骁龙 765G 又怎样？我不玩游戏够用了；屏幕挖孔又怎样？反正看久了会习惯的。

于是放假后我就找了个家教的工作，攒钱换手机。当然按照赚钱进度，今天是不可能拿到手机的，但是前天中午吃饭的时候，母上突然问我想买的手机多少钱？然后就提前帮我先买了，虽然是预支，但能提前摸到手机我已经很开心了

开箱

和之前一样的包装，小巧的盒子，打开后是手机，说明书，充电线，还有个转接头。

手机比想象中更精致，对比我的 Pixel 2 XL 小了一圈，后盖是磨砂的，摸着很舒服

解锁

到手第一件事自然是解锁 & Root 了，初次开机后跳过全部东西，然后进设置，点版本号打开开发者选项，结果 oem 的开关是灰的。

当时真的把我吓到了，以为被淘宝奸商坑了，然后看了下面提示说联网，还好，联完网就好了，打开开关，关机

关机后，音量下和电源键同时按住进入 Bootloader 界面

上手体验

到手几个小时了，90Hz 感觉不大出来，可能是电脑显示器 144 Hz 已经看习惯了的缘故。

手感上就是很小巧，单手完全没问题，网上说的屏幕泛绿感觉有点，不知道是不是心理作用。

其它的可能就要后面感受了，之后可能会写篇详细的使用体验，也可能不会（现在感觉没啥好写的东西），看情况吧。
