记第一次vps被攻击

2020-04-01 破事水

文章目录

本文最后更新于 <span id="expire-date"></span> 天前，文中部分描述可能已经过时。

早上起来发现我的大盘鸡流量不正常

一直在上传，在cloudflare一看一晚上跑了近500G流量

幸好装了探针，要不然根本发现不了

使用 iptop 命令一看，果然有几个不正常的ip

果断把他们ban了

iptables -A INPUT -s ip -j DROP

马上流量就正常了

还好问题不是太大，要不然我还真不知道怎么办，这还是我存数据的鸡，可不能随随便便重装
